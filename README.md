# Slot Machine for KDE Plasma 6

casino slot machine widget for KDE Plasma 6 desktop.

## Installation

### From Source
```bash
git clone https://github.com/blueberriesflower/plasma-slot-machine.git
cd plasma-slot-machine
mkdir -p ~/.local/share/plasma/plasmoids/org.kde.onedarmbandit
cp -r . ~/.local/share/plasma/plasmoids/org.kde.onedarmbandit/
